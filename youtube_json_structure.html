How to navigate this in Python:

payload['items'][0]['id'] → channel ID (use in next request)
payload['items'][0]['snippet']['title'] → channel name
payload['items'][0]['snippet']['publishedAt'] → date channel was created
payload['items'][0]['snippet']['thumbnails']['high']['url'] → high-res thumbnail URL
